xFrameworkPX Release Notes for 3.5.4

Release Date: 2010/12/16
Current Version: 3.5.4
Previous Version: 3.5.2


Overview

  • consoleの不具合を修正
    • webapp/index.htmlがないとエラーとなってしまう不具合を修正しました。
    • 対象コントローラがない場合にエラーにならない不具合を修正しました。
  • isIE9の実装
    • テンプレート上でisIE9が使用できるようになりました。
  • isValidの改修
    • 引数を保持するプロパティを設置しました。
    • ユーザーバリデーション内で、上記プロパティを取得できるgetTargetDatasメソッドを実装しました。

Change Log

xFrameworkPX Ver. 3.5.2
  • ディレクトリ構成
    • controllersディレクトリが消えていたので、追加しなおしました。
  • クラスのオートロード
    • 継承元クラスを自動的にロードする機能にあったバグを修正しました。
  • WiseTag
    • editメソッドにあったバグを修正しました。
  • データベース接続処理
    • 指定された文字コードが正しく設定されていなかったバグを修正しました。
  • ドキュメント
    • ドキュメントを大幅に加筆修正しました。
    • チュートリアルを大幅に加筆修正しました。
xFrameworkPX Ver. 3.5.1
  • STRICTモード対応
    • エラーレベルがSTRICTモードで実行したとき、xFrameworkPX本体でエラーが起こらないように対応しました。
  • データベース接続処理
    • ポート番号が指定できるようになりました。
    • データベース接続時に例外が発生したとき、表示されていたExceptionの画面で出ていたNoticeに対応しました。
  • LiveRecord
    • getメソッドのwhere句を指定するパラメータで、「IS NULL」と「IS NOT NULL」が指定できるようになりました。
    • LiveRecordのメソッド実行時にデータベースに接続されていなかった場合、Exceptionが発生するようになりました。
  • WiseTag
    • 入力フォームを動的に生成する「WiseTag」の機能が実装されました。
  • Library
    • xFrameworkPXに組み込まれているPDFを操作するためのライブラリとExcelファイルを操作するためのライブラリをアップデートしました。
  • RapidDrive
    • WiseTagを組み込むため、大幅に仕様を変更しました。
    • ver 3.5.0までにあったバグや問題を改修しました。
  • 拡張メソッド
    • base_nameメソッドにポート番号を取得する機能が追加されました。
    • SSL接続判定メソッドにあったバグを修正しました。
xFrameworkPX Ver. 3.5.0
  • ディスパッチャー
    • コントローラーがない場合でも、スーパーアクション、グローバルアクションのみを実行させることができるようになりました。
  • クラスのオートロード
    • コントローラーとモジュールクラスの継承もとクラスを、自動的にロードする機能が追加されました。
  • データベース接続処理
    • 接続設定を切り替えられるようになりました。
    • データベースの接続設定が未指定時に、xFrameworkPXが自動的に選択する接続設定を指定できるようになりました。
    • データベースの接続設定が複数あった場合に、設定を正しくロードできていなかったバグを修正しました。
  • LiveRecord
    • getメソッドでSQLの関数が使えるようになりました。
    • getメソッドにGROUP BY句、HAVING句の指定をするためのパラメータが追加されました。
    • LiveRecordがOracleに対応しました。
    • LiveRecordがPostgreSQLに対応しました。
xFrameworkPX Ver. 3.5.0 Release Candidate
  • xFrameworkPX Debug Tools 1.0.2
    • セッションタブを追加しました。
    • クッキータブを追加しました。
    • コンテンツにより表示されないバグを修正しました。
  • xFrameworkPX 3.5 Documentation
    • ダイレクトリンク機能を実装しました。
    • ローディングCSSを修正しました。
    • コンテンツの追加＆修正しました。
xFrameworkPX Ver. 3.5.0 Beta
  • ディスパッチャー
    • 正規化されたURLで無い場合は、正規化してリダイレクトするようになりました。
    • パスの末尾が/でない場合で404エラーになる場合は、/を付加してリダイレクトするようになりました。
  • コントローラー

      • 引数に渡す値をxFrameworkPX_Util_MixedCollection以外に配列でも指定できるようになりました。
    • xFrameworkPX_Controller_Action
      • RapidDriveのmodule指定を行わない場合、モジュール指定されている一番最初のモジュールが自動的に利用されるようになりました。
    • xFrameworkPX_Controller_Component_RapidDrive
      • detailモード時、listqueryをユーザーデータとして自動的に登録するようになりました。
        このパラメータは、詳細画面表示時に渡される検索文字列とページャー番号を含んでいます。
        list画面へ戻る際のリンクに使用します。
